Reading strategy

1. Find the class definition — what data does it hold?
2. Find the main update loop (update(), run(), execute())
3. Read only that loop first — ignore everything else
4. Trace one specific behavior end to end
5. Write down one difference from the tutorial version and why
The goal is not to understand everything — it's to find the same patterns (PID, state machines, publishers) in production code. Each match builds pattern recognition.
